# 📱 CalculatorApp

🚀 Um aplicativo de **Calculadora** desenvolvido em **Kotlin** com **Jetpack Compose**.

## 📌 Tecnologias Utilizadas

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Android Studio**

---

## 🎯 Funcionalidades

✔️ Interface moderna com **Jetpack Compose**  
✔️ Suporte a **operações básicas** (+, -, ×, ÷)  
✔️ Exibição clara do cálculo  
✔️ Layout responsivo
